# Teledyne 2026 Leadership Development Program - Coding & Software Engineering

**Sponsor:** Brian Maguire, Executive VP Marine US

This organization supports the 2026 LDP Coding and Software Engineering cohort. We're building engineering practices, shared knowledge, and reusable code across Teledyne's business units.

## Program Focus

- **Knowledge sharing** for GitHub Copilot and reusable code
- **Code quality benchmarking** with AI assistance
- **Bootcamps** for beginning, intermediate, and advanced developers

## Getting Started

1. Review the [Code Review Guidelines](link-to-repo)
2. Join the LDP Coding Slack/Teams channel (link TBD)
3. Complete your first project assignment (see Projects tab)

## Approved Tools

- GitHub Copilot (enterprise license)
- GitHub Actions for CI/CD
- Branch protection and code review required

## Governance

All code and documentation must follow [Teledyne AI Governance](internal-link). No ITAR, ECI, or CUI in repos. Contact Matt Langston (AI & Platform Engineering) for questions.
